# Panduan Migrasi dan Seeding Modul Pelatihan

Karena seluruh _file_ migrasi dan _seeding_ untuk modul Pelatihan telah dipisahkan ke dalam _folder_ khusus (`app/Database/Migrations/Pelatihan` dan `app/Database/Seeds/Pelatihan`), berikut adalah daftar perintah _Command Line_ (CLI) CodeIgniter 4 yang tepat untuk menjalankannya. Panduan ini dapat Anda sertakan ke dalam dokumentasi repositori.

## 1. Menjalankan Migrasi
CodeIgniter 4 secara pintar akan memindai semua _sub-folder_ di dalam direktori `Migrations` selama _namespace_-nya tetap `App\Database\Migrations`.

Oleh karena itu, perintah migrasi untuk mengeksekusi tabel pelatihan **sama persis** dengan perintah *default* CI4:
```bash
# Menjalankan semua migrasi yang belum tereksekusi
php spark migrate

# Mengecek status migrasi (melihat batch migrasi)
php spark migrate:status

# Me-reset ulang (drop semua tabel) dan menjalankan migrasi dari awal
php spark migrate:refresh
```

## 2. Menjalankan Seeding Khusus Pelatihan
Karena *file* *Seeder* Pelatihan berada di dalam *namespace* khusus (`App\Database\Seeds\Pelatihan`), Anda harus menyertakan nama *namespace* dan *class*-nya secara penuh di dalam perintah `spark` (diapit tanda kutip):

```bash
# Mengeksekusi seeder utama (DataAwalPelatihanSeeder) yang otomatis akan memanggil seeder pelatihan lainnya
php spark db:seed "App\Database\Seeds\Pelatihan\DataAwalPelatihanSeeder"
```

> [!TIP]
> **Catatan Penting Penggunaan Tanda Kutip (`"`)**
> Pada sistem operasi Windows (terutama jika menggunakan PowerShell), pemanggilan *namespace* yang mengandung garis miring terbalik (*backslash* `\`) harus dibungkus dengan tanda kutip dua ganda (`"..."`) agar karakter `\` tidak diabaikan/di-*escape* oleh terminal.

## 3. Menjalankan Seeder Tertentu secara Spesifik
Jika Anda hanya ingin menjalankan satu *seeder* tertentu (bukan semuanya lewat `DataAwalPelatihanSeeder`), Anda juga harus menyertakan _namespace_ lengkapnya:

```bash
# Contoh: Hanya menyemai data Users (Akun)
php spark db:seed "App\Database\Seeds\Pelatihan\UsersPelatihanSeeder"

# Contoh: Hanya menyemai data Master Pelatihan
php spark db:seed "App\Database\Seeds\Pelatihan\MasterPelatihanSeeder"
```
